# Frequently Asked Questions

The context menu does not appear next to the mouse click ?

> Make sure you imported Angular CDK overlay CSS. [See Installation and Setup](?path=/story/context-menu-installation-and-setup--page#import-the-angular-cdk-overlay-css)

Where this library comes from ?

> This project has been initially forked from [ngx-contextmenu](https://github.com/isaacplmann/ngx-contextmenu.git) to port it to Angular 13.
> It was originally inspired by [ui.bootstrap.contextMenu](https://github.com/Templarian/ui.bootstrap.contextMenu).

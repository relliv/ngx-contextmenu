# ContextMenuService

The `ContextMenuService` is an Angular service that can be injected in your component or service.
It provides:

- a `closeAll` method that will close any open context menu
- a `hasOpenMenu` to check if there is at least one context menu opened
- a `show` method to programmatically open a context menu, but it mots cases you would prefer to use the open method of the context menu directive

| `ContextMenuService` | Parameters                                                                                                      | Description                                                                                                                    |
| :------------------- | :-------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------- |
| `closeAll()`         | -                                                                                                               | Close all open context menus                                                                                                   |
| `hasOpenMenu`        | -                                                                                                               | Check if there is at least one context menu opened                                                                             |
| `show`               | - contextMenu: ContextMenuComponent<br />- options: ContextMenuOpenAtPositionOptions (default: \{ x: 0, y: 0 }) | Programmatically open a context menu, but in most cases you should prefer to use the open method of the context menu directive |
